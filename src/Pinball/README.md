# Ball of Duty

## Description
This game is a pinball game in neon colors built with Java Swing. The user is first greeted by an authentication screen where players register or login through a database in `users.txt`. When users login, they have an interactive display where they can see their score, coins, lives, active powerup, as well as the speed of the bar. This game is built with an in-house physics engine which responds to the various actions. Each bumper hit awards a various number of points and the user’s score is automatically saved. After each round, the player can buy powerups in the store which allows them to earn even more money.

## How to Run the Program
To run the program, all files in this repository will have to be downloaded through GitHub. When all files are downloaded, the user will need to run the `AuthManager.java` file which contains the main method.
