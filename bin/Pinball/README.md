Game Name: Ball of Duty
Description: This game is a pinball game in neon colors built with Java Swing. The user is first greeted by an authentication screen where players register or login through a database in users.txt. When users login, they have an interactive display where they can see their score, coins, lives, active powerup, as well as the speed of the bar. This game is built with an in-house physics engine which responds to the various actions. Each bumper hit awards a various number of points and the userès score is automatically saved After each round, the player can buy powerups in the store while allows them to earn even more money.

Prompt: develop java-swing pinball with key controls in four files without errors with numerous classes and files with proper documentation and make it so hitting the bumpers gives you money which can be exchanged for new powerups, create the store within, incorporate both authentication also score-saving with text files access

How to Run the Program:
To run the program, all files in this repository will have to be downloaded through GitHub. Whel all files are downloaded, the user will need to run the AuthManager.java file which enhouses the main method.

Program Goal:
The goal of this game is to display my understanding of GUI as well as several other minor topics through a fun format, I was able to utilize my understanding to build a game which I am proud of. The end goal of this game is to maximize happiness while reducing the number of bugs encountered.
