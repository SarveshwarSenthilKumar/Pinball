Ball of Duty is a pinball game developed in Java Swing in order to demonstrate concepts learned in the GUI unit of our curriculum. These concepts include Collision Detection, Mouse and Button Events (Keyboard), Object-Oriented Programming through Abstract Classes and Inheritance, GUI Design, as well as manipulation of files or Files I/O. Firstly, collision detection is used throughout the program as this is an action-based collision style game where each collision between walls, bumpers, and flippers corresponds to different results. The main control mechanism for the game is through the arrow keys which requires keyboard input and detection, mouse events are also necessary in responding properly to the user’s intentions when it comes to the store and other clickable buttons. Abstract classes and object-oriented programming theory and logic is used throughout the whole program in order to ensure a more comprehensive program and a more understandable logic. GUI design is used throughout the program and is the core underpinnings of the app. Files and I/O are used with relation to authentication as well as for keeping track of score for future reference once the game is closed.
